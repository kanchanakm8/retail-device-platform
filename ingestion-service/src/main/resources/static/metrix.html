How to read this dashboard

Processed Events should increase when valid events are successfully consumed by inventory-service.

Failed Events increase when downstream processing fails.

Duplicate Events increase when the same eventId is received again and idempotency blocks it.

Published Success / Failure / DLQ reflect the publisher-side behavior in ingestion-service before and during delivery to inventory-service.
